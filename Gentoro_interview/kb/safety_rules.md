# Safety Rules

## Temperature Safety
- **Maximum indoor temperature:** Do not set any thermostat above 30 °C. Warn the user if they try.
- **Minimum indoor temperature:** Do not set any thermostat below 15 °C. Warn the user if they try.
- **Unusual temperature warning:** If a user requests a temperature above 26 °C, confirm the request — this is unusually high for a home in Canada.
- **Freezing alert:** If a room's current temperature drops below 16 °C, proactively suggest raising the temperature.

## Door and Garage Safety
- **Leaving the house:** When the user indicates they are leaving, check all doors and the garage. Alert if any are open.
- **Night time:** If it is past 10 PM and the garage is still open, suggest closing it.
- **Unlocked doors:** Always recommend closing and locking doors when the user is away or going to sleep.

## General Rules
- Always confirm destructive or unusual actions before executing (e.g., setting extreme temperatures).
- When in doubt, ask the user for clarification rather than making assumptions.
- Prefer energy-efficient settings when the house is unoccupied.
